Amazon Reviews Sentiment Analysis

This project utilizes TensorFlow to develop a simple and efficient sentiment analysis model for Amazon customer reviews. By specifying the input shape, output shape, and essential layers, the model automatically infers intermediate layer sizes, streamlining the process.

The project focuses on text preprocessing, utilizing techniques such as tokenization, stopword removal, and embedding to efficiently handle text data. A modular design allows for easy experimentation with different layers, including CNN, LSTM, and Dense, to achieve accurate sentiment classification.

The model is optimized for efficient training and inference, with automatic hyperparameter tuning to enhance performance. The project aims to achieve high accuracy in sentiment classification, exploring the impact of different architectures on performance.

While similar to Keras's Sequential model framework, this project offers a customized approach to sentiment analysis. Future enhancements include incorporating attention mechanisms and aspect-based sentiment analysis.

The project is built using Python, TensorFlow, NLTK, spaCy, and scikit-learn, leveraging the publicly available Amazon Review Dataset. For more information and code details, visit the GitHub Repository Link. Related projects can be found at Link to related projects.
